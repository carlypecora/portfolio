# carlypecora.com

Personal portfolio site. Built with React 18 and deployed on Netlify.

## Prerequisites

- Node.js >= 18
- Yarn

## Setup

```bash
yarn install
```

## Development

```bash
yarn start
```

Opens at [http://localhost:3000](http://localhost:3000). Hot-reloads on save.

## Production Build

```bash
yarn build
```

Output goes to `build/`. Deployed automatically via Netlify.
